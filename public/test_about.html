API

プロフィール情報をJSON形式で返します。

プロフィール一覧API

URL

http://makesthis.com/profiles.js

パラメーター

例:http://makesthis.com/profiles.js?page=2&per_page=20

page
ページ数。デフォルトは1。
per_page
1回のリクエストに含めるプロフィール数。デフォルトは100。(最大100)
sort
並び順。選べる項目はid, created_at, updated_at。デフォルトはid。
order
並び順の昇順・降順。選べる項目はasc, desc。デフォルトはasc。
