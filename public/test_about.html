My name is Masaki Komagata. I work as a programmer at a company in Hatsudai called FJORD, LLC. Before that, I hopped around between various internet start-up companies.

駒形真幸(コマガタマサキ)と申します。初台にあるFJORD, LLCという会社でプログラマーとして働いています。それまでは色々なネットベンチャーを点々としていました。

何を作りましたか？

I've made a CMS tool called Lokka. It's a blogging/CMS tool that's written in Ruby. It's similar to WordPress and functions in a Heroku, Google App Engine cloud environment.

Lokka(ロッカ)というCMSを作りました。これはRubyで書かれたHeroku, Google App Engineといったクラウド環境にも対応したWordPressライクなブログツール・CMSです。

何故それを作りましたか？

Because Wordpress doesn't function in a cloud environment. I really love Heroku, this free, simple cloud (PaaS) service. I made Pyha because I wanted something similar to WordPress that operates in Heroku.

WordPress is useful and I like it a lot. But I used to get annoyed with the lack of consistency in the API. In the summer of 2010 after finishing a long-term project using WordPress, I decided to spend my entire summer vacation redesigning WordPress using Ruby.

WordPressがクラウド環境で動かないからです。僕は無料で簡単なクラウド(PaaS)サービスのHerokuが大好きです。Herokuで動くWordPress的なモノが欲しかったので作りました。

WordPressは便利で大好きです。しかし一方でAPIの統一感の無さに腹を立てていました。2010年の夏、WordPressを使った長い仕事が終わったとき、Rubyを使ってWordPressをリデザインすることに夏休みを全て費やす決意をしました。

次に何を作りますか？

I'd like to make an online AI-based game that runs on the browser of a PC or a smart phone. So I'm currently studying Javascript, WebSockets, WebGL, and making prototypes.

PCやスマートフォンのブラウザ上で動く、AIがプログラミングできるネットゲームを作りたいと思っています。そのためのJavascript, WebSockets, WebGLを勉強したり、プロトタイプを作ったりしています。
